# 📘 Техническое задание  
## Проект: «Математический тренажёр»

---

## 1. Общая идея  
Сайт для практики математики с адаптивным подбором задач, автоматической проверкой, системой прогресса и встроенным ИИ-помощником.

Сайт должен:
- подбирать задачи по выбранным теме и уровню  
- объяснять ошибки  
- отслеживать прогресс  

---

## 2. Платформа и технологии  
**Backend:**  
- Python (flask)

**База данных:**  
- sqlalchemy  


**ИИ-помощник:**  
- huggingface API  
---

## 3. Основной функционал  
- Система задач

- Разделение по:
  - темам (алгебра, геометрия и т.д.)
  - уровням сложности (1–10)

## 4. ИИ-помощник  

**Функции:**
- подсказки  
- объяснение решения  
- разбор ошибок

---

## 5. Достижения  

**Примеры:**
- 10 задач подряд без ошибок  
- 100 решённых задач  

---

## 6. Хранение данных
- База данных sqlite, управляемая через sqlalchemy


## Перед использованием:
  - pip install -r requirements.txt
  - для использования ии нужен HF_API_KEY в .env
