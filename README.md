# 📘 Техническое задание  
## Проект: Telegram-бот «Математический тренажёр»

---

## 1. Общая идея  
Telegram-бот для практики математики с адаптивным подбором задач, автоматической проверкой, системой прогресса и встроенным ИИ-помощником.

Бот должен:
- подбирать задачи по выбранным теме и уровню  
- объяснять ошибки  
- отслеживать прогресс  

---

## 2. Платформа и технологии  

**Платформа:**  
- Telegram Bot API  

**Backend (один из вариантов):**  
- Python (aiogram)  

**База данных:**  
- sqlalchemy  


**ИИ-помощник:**  
- huggingface API  

---

## 3. Основной функционал  
- Статистика

- Система задач

- Разделение по:
  - темам (алгебра, геометрия и т.д.)
  - уровням сложности (1–10)


**Процесс:**
1. Бот отправляет задачу  
2. Пользователь отвечает  
3. Бот проверяет  
4. Бот даёт фидбек  


## 4. ИИ-помощник  

**Функции:**
- подсказки  
- объяснение решения  
- разбор ошибок  


**Ограничение:**
- не выдаёт полный ответ без запроса  

---

## 5. Достижения  

**Примеры:**
- 10 задач подряд без ошибок  
- 100 решённых задач  
- завершение темы  

**Команда:**
- `/achievements`

---

## 6. Хрвненте данных
- База данных sqlite, управляемая через sqlalchemy
